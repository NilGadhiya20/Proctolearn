## 🚀 How to Use (Quick Start)

### 1. Start Backend
```bash
cd backend
npm run dev
# Runs on http://localhost:5000
```

### 2. Start Frontend
```bash
cd frontend
npm run dev
# Runs on http://localhost:3000 or 5173
```

### 3. Test It
- Go to `http://localhost:3000/subscribe`
- Enter your email
- Submit the form
- Check your inbox for confirmation email

| Issue | Solution |
|-------|----------|
