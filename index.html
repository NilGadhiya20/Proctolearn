🚀 Quick Start

1. Setup Backend

cd backend && npm install && npm run dev

Runs on http://localhost:5000

2. Setup Frontend

cd frontend && npm install && npm run dev

Runs on http://localhost:3000

3. Configure .env

cp .env.example .env

Add your configuration details

4. Start MongoDB

net start MongoDB

Windows command (see docs for other OS)

5. Access App

http://localhost:3000

Open in your browser

6. Login

Create account or use test credentials

See INSTALLATION.md for test credentials
